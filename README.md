# Dogan Can Yucel — Personal Website

Personal portfolio website for [dogancanyucel.com](https://dogancanyucel.com)

## Stack
- Pure HTML5, CSS3, Vanilla JS (no dependencies, no build step)
- Google Fonts: Outfit + JetBrains Mono

## Features
- ✨ Animated particle canvas hero
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎯 Scroll-triggered animations
- 📊 Animated skill bars
- 📬 Contact form (Formspree)
- 🌙 Dark theme with turquoise accents
- 🔍 SEO optimized

## Deployment: Cloudflare Pages

1. Push this folder to a **GitHub repository**
2. Go to **Cloudflare Dashboard → Pages → Create a project**
3. Connect your GitHub repo
4. Build settings: leave empty (static site, no build command needed)
5. Add custom domain: `dogancanyucel.com`

## Contact Form Setup (Optional)
1. Go to [formspree.io](https://formspree.io) → Create free account
2. New form → Get your Form ID (e.g. `xaybzqkd`)  
3. In `index.html`, replace `YOUR_FORM_ID` with your actual ID
